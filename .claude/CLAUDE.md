## Python coding style
* Always follow PEP 8
* Use indentation by 4 spaves per indentation level
* Maximum line lengh is 120 characters
* Always write docstrings to functions using reST syntax
* Always write type hints
* Write explanation comment to all regular expressions
* Follow Keep It Stupid Simple (KISS) principle
* Use modern Python syntax ond constructions, targeting Python 3.14

### Qt
* For Qt elements, use PySide6 targeting Qt 6.11
